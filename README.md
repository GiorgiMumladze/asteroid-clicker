💬 Asteroid Clicker
Built with React and Vite
Fast-paced clicker game with smooth UI animations
Random asteroid spawning system with increasing difficulty
Accurate score tracking based on clicks per session
Multiple game modes (15s / 30s / 60s timer options)
LocalStorage support for saving best scores per mode
Sound effects + optional background music toggle
Responsive design optimized for desktop and mobile
Clean React hooks state management (useState, useEffect, useRef)
Game-over screen with final score + restart/quit options
